# 阶段 3 — 面向对象深入

> 目标：掌握 Python 独有的 OOP 特性（描述符、MRO、Protocol）。

## 计划内容

- 类基础、`__init__` / `__new__`
- 继承、MRO（C3 线性化）、`super()`
- 多继承与菱形问题
- `property` 描述符协议
- `__getattr__` / `__setattr__` / `__getattribute__`
- 描述符深入：方法、`property`、`staticmethod` / `classmethod` 真相
- `__slots__` 与内存优化
- ABC（`abc.ABCMeta`） vs `typing.Protocol`（结构化子类型）

## 项目

可扩展的规则引擎：用 `Protocol` 定义接口、注册插件。

## 启动条件

阶段 2 项目通过。
